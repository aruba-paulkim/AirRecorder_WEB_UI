------------------------------------------------------------------------------
https://github.com/aruba-paulkim/AirRecorder_WEB_UI

------------------------------------------------------------------------------
### Starting Service
------------------------------------------------------------------------------
<pre>git clone https://github.com/aruba-paulkim/AirRecorder_WEB_UI.git
cd AirRecorder_WEB_UI
npm install
npm start</pre>
connect to http://127.0.0.1:8900
